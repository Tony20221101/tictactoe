# Tic Tac Toe

A simple, interactive web-based Tic Tac Toe game built with HTML, CSS, and JavaScript.

## Features

- Two-player gameplay (X and O)
- Score tracking
- Visual animations for winning moves
- Responsive design
- Clean, modern UI

## How to Play

1. Open `tictactoe.html` in your web browser
2. Click on any cell to place your mark
3. First player to get 3 in a row wins!
4. Click "New Game" to reset the board

## Running Locally

You can serve the game using any static file server:

```bash
python3 -m http.server 8080
```

Then open http://localhost:8080/tictactoe.html
